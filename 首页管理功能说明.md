## 🚀 使用指南

### 访问后台
1. 浏览器访问：`http://localhost:3000/#admin`
2. 点击顶部"**首页管理**"标签

### 轮播图管理
1. 点击"**添加轮播图**"按钮
2. 填写标题、副标题、图片URL
3. 选择背景渐变色
4. 设置按钮文字和跳转链接
5. 点击"**添加轮播图**"保存

**管理操作：**
- **启用/禁用**：快速控制显示状态
- **上移/下移**：调整轮播顺序
- **编辑**：修改内容
- **删除**：永久删除（需确认）

### 特性功能管理
1. 点击"**添加特性**"按钮
2. 输入图标类名（如：`fa-truck-fast`）
3. 填写标题和描述
4. 设置排序和状态
5. 保存

**图标查找：**
访问 [fontawesome.com/icons](https://fontawesome.com/icons) 查找图标

### 品牌故事管理
1. 点击"**编辑内容**"按钮
2. 填写所有表单字段
3. 上传或填写背景图片 URL
4. 配置3组统计数据
5. 设置2个CTA按钮
6. 保存修改

### 视频展示管理
1. 点击"**添加视频**"按钮
2. 填写视频标题
3. 上传或填写封面图 URL
4. 设置产品标签和观看次数
5. （可选）填写视频URL
6. 保存

### Newsletter 管理
1. 点击"**编辑内容**"按钮
2. 自定义所有文案
3. 选择背景渐变色
4. 配置3个特性徽章
5. 保存修改

## 📸 图片资源推荐

### 免费高质量图库：
1. **Unsplash** (https://unsplash.com) ⭐
   - 高质量免费图片
   - 可直接使用URL
   - 支持URL参数优化

2. **Pexels** (https://pexels.com)
   - 免费可商用
   - 视频素材丰富

3. **Pixabay** (https://pixabay.com)
   - 免费素材库
   - 多语言支持

### Unsplash URL 优化技巧：
```
原始链接：
https://images.unsplash.com/photo-xxxxx

优化链接（添加参数）：
https://images.unsplash.com/photo-xxxxx?auto=format&fit=crop&q=80&w=1400

参数说明：
- auto=format：自动格式化
- fit=crop：裁剪适配
- q=80：质量80%
- w=1400：宽度1400px
```

## 💡 最佳实践

### 内容设计
1. **标题要简洁有力**
   - 突出核心卖点
   - 控制在10字以内
   - 使用行动导向语言

2. **描述要清晰明确**
   - 补充说明价值
   - 增加可信度
   - 控制在50字左右

3. **图片要高质量**
   - 使用专业摄影图片
   - 确保清晰度
   - 保持风格统一

### 数量建议
- **轮播图**：3-5个
- **特性功能**：4-6个
- **视频展示**：3-8个

### 排序策略
1. 最重要内容放前面
2. 根据数据调整顺序
3. 保持逻辑连贯性

### 图片优化
1. **压缩工具：**
   - TinyPNG (https://tinypng.com)
   - Squoosh (https://squoosh.app)

2. **尺寸规范：**
   - 严格按照推荐尺寸
   - 使用2倍分辨率（Retina）
   - 保持宽高比

3. **格式选择：**
   - 照片用 JPG
   - 图标/Logo 用 PNG
   - 动画用 GIF/WebP

## 🆘 常见问题

### Q1: 如何找到合适的图标？
**A:** 访问 [fontawesome.com](https://fontawesome.com/icons)，搜索关键词，复制类名（如 `fa-truck-fast`）

### Q2: 图片加载很慢怎么办？
**A:** 
- 压缩图片至推荐大小
- 使用 Unsplash 的 URL 参数优化
- 考虑使用 CDN

### Q3: 修改后前台没有更新？
**A:** 
- 刷新前台页面
- 清除浏览器缓存
- 检查是否已保存修改

### Q4: 如何恢复默认设置？
**A:** 
- 手动编辑 App.tsx
- 恢复初始数据
- 或重新部署项目

### Q5: 支持移动端管理吗？
**A:** 
- 当前界面响应式设计
- 支持平板和手机访问
- 推荐使用桌面端操作
